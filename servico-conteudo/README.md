# Sistema de Entrega — Gestor de Conteúdo Anual

## Oferta
- **Anual parcelado:** R$25.000/ano
- **Anual à vista:** R$20.000/ano
- **Bônus inclusos:** Auditoria + Posicionamento completo

## O que o cliente recebe por mês
| Entregável | Quantidade |
|-----------|-----------|
| Carrosseis prontos (Canva editável) | 8 |
| Scripts de Reels | 4 |
| Sequências de Stories | 8 |
| Calendário editorial | 1 |
| **Total de peças/mês** | **20** |
| **Total de peças/ano** | **240+** |

## Fluxo completo
```
VENDA FECHADA
    ↓
01-ONBOARDING → formulário de briefing (cliente preenche)
    ↓
02-AUDITORIA → você roda o squad (30 min) → entrega relatório PDF
    ↓
03-POSICIONAMENTO → você roda o squad (1h) → entrega bio + CLC + StoryAds
    ↓
04-MENSAL → todo mês você roda o squad (3h) → entrega 20 peças no Drive
    ↓
05-ENTREGA → pasta organizada no Google Drive do cliente
```

## Pastas deste sistema
- `01-onboarding/` — Formulário e checklist de entrada
- `02-auditoria/` — Processo e template do relatório
- `03-posicionamento/` — Processo e template de entrega
- `04-mensal/` — Processo e template mensal
- `05-entrega/` — Modelo de pasta de entrega para o cliente
