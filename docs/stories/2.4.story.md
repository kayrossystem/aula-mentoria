## Contexto

Com a interface de hook projetada na Story 2.3, esta story implementa os adapters para as 2 IDEs de maior uso: Claude Code e Cursor. As outras 3 IDEs (Codex, Gemini, AntiGravity) são planejadas para o sprint seguinte.

O objetivo é validar que a interface abstrata é implementável na prática antes de migrar todas as IDEs.
