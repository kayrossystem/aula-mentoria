## Contexto

MCP custom presets é uma funcionalidade existente que está completamente invisível para o usuário. Não há documentação sobre como criar, configurar ou usar presets customizados. Resultado: usuários criam workarounds manuais para algo que o sistema já resolve nativamente.

Esta é a story com menor esforço e impacto imediato em usabilidade — pode ser executada independentemente de qualquer outro epic.
