# Checklist: Oraculo Posts

> Validacao obrigatoria antes de entregar qualquer carrossel.
> Score >= 80% para aprovar.

---

## Slide 1 (Hook)

- [ ] Max 15 palavras
- [ ] Sem pergunta (interrogacao)
- [ ] Contraintuitivo
- [ ] Zero contexto, so tensao
- [ ] 3 variacoes geradas (2 virais + 1 imperial)
- [ ] Formato S1_HOOKS correto

## Slides 2-10

- [ ] Max 30 palavras por slide
- [ ] Cada slide adiciona informacao nova
- [ ] Progressao emocional: Reptiliano (1-3) -> Limbico (4-6) -> Neocortex (7-10)
- [ ] Tensao constante do inicio ao fim

## Linguagem

- [ ] Sem palavras proibidas (cliches-proibidos.md)
- [ ] Sem frases de rejeicao imediata
- [ ] Sem cliches de coach
- [ ] Foco no leitor ("voce"), nunca "eu/nos"
- [ ] Tom NUCLEO Torriani mantido

## Estrutura

- [ ] Tipo de post correto (estrutura slide-a-slide)
- [ ] Framework de copy aplicado
- [ ] Solucao nao revelada cedo demais
- [ ] Pelo menos 1 dos 5 efeitos obrigatorios presente
- [ ] Arco dramatico com climax e decisao

## CTA

- [ ] 3 opcoes geradas
- [ ] Sem "clique aqui", "me chama", "link na bio" (generico)
- [ ] Comando moral + palavra-chave + promessa
- [ ] Alinhado com intencao do post

## 12 Testes do Validator

- [ ] 1. Autenticidade Emocional
- [ ] 2. Ausencia de Marcadores
- [ ] 3. Presenca Cinematografica
- [ ] 4. Linguagem Cirurgica
- [ ] 5. Profundidade Experiencial
- [ ] 6. Comando Militar
- [ ] 7. Tensao Narrativa
- [ ] 8. Valor x Brutalidade (70/30)
- [ ] 9. Estrutura Integrada
- [ ] 10. Variabilidade Expressiva
- [ ] 11. Coerencia com Frameworks
- [ ] 12. Protecao Contra Vazamentos

## Resultado

- [ ] Score >= 80%: APROVADO
- [ ] Score < 80%: REESCREVER
