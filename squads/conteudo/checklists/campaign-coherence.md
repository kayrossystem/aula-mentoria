# Checklist de Coerencia de Campanha Multi-Formato

## MENSAGEM CENTRAL
- [ ] Mensagem-chave unica e clara definida
- [ ] Mensagem consistente entre todos os formatos (Feed, Stories, Reels)
- [ ] Nenhum formato contradiz ou enfraquece a mensagem central
- [ ] Adaptacao de linguagem por formato sem perder a essencia

## PROGRESSAO EMOCIONAL
- [ ] Jornada emocional planejada entre as pecas
- [ ] Progressao coerente (tensao -> desenvolvimento -> resolucao)
- [ ] Cada peca contribui para o arco narrativo geral
- [ ] Nao ha saltos emocionais bruscos ou desconexos

## TOM IMPERIAL
- [ ] Tom imperial mantido no Feed
- [ ] Tom imperial mantido nos Stories
- [ ] Tom imperial mantido nos Reels
- [ ] Tom imperial mantido nos CTAs
- [ ] Sem quebra de registro (formal/informal inconsistente)
- [ ] Autoridade presente em todas as pecas

## CTAs ALINHADOS
- [ ] CTAs direcionam para o mesmo objetivo final
- [ ] CTAs se complementam entre formatos
- [ ] Progressao logica dos CTAs (awareness -> interesse -> acao)
- [ ] Nenhum CTA contradiz outro
- [ ] Palavra-chave consistente nos CTAs

## CRONOGRAMA INTEGRADO
- [ ] Feed e Stories publicados em horarios complementares
- [ ] Sequencia de publicacao segue a progressao narrativa
- [ ] Espacamento adequado entre pecas
- [ ] Stories amplificam/referenciam posts do Feed
- [ ] Nenhuma sobreposicao que canibalize atencao

## CONSISTENCIA VISUAL (quando aplicavel)
- [ ] Paleta de cores coerente entre formatos
- [ ] Estilo visual reconhecivel como mesma campanha
- [ ] Elementos visuais se complementam

## TESTE FINAL
- [ ] Se alguem visse apenas 1 peca, entenderia a mensagem?
- [ ] Se alguem visse todas as pecas, a experiencia e fluida?
- [ ] Nenhuma peca parece "solta" ou desconectada?
- [ ] A campanha como um todo e maior que a soma das partes?
