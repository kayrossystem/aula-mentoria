# Checklist: Oraculo Reels

> Validacao em 3 niveis. Score >= 80% em CADA nivel.
> Progressao obrigatoria: Nivel 1 -> 2 -> 3.

---

## Nivel 1: Copy

- [ ] Beneficio forte e irresistivel
- [ ] Track record / prova presente
- [ ] Credibilidade crivel
- [ ] Angulo diferenciador (ideia unica)
- [ ] Util, Urgente, Unico, Ultraespecifico
- [ ] Especificidade brutal (numeros exatos, impares, contexto)
- [ ] 12 Testes do Validator aprovados
- [ ] Sem palavras proibidas (compliance 100%)
- [ ] Score Nivel 1 >= 80%

## Nivel 2: Hook

- [ ] Usa 1+ dos 7 padroes virais
- [ ] Max 5 palavras (>5 = reprovacao automatica)
- [ ] Primeira palavra inesperada (nao "Voce", "Eu", "A", "O")
- [ ] Texto na tela + falado
- [ ] Gera sentimento forte (Choque/Curiosidade/Raiva/Medo)
- [ ] Nao revela solucao (incompletude forcada)
- [ ] Score Nivel 2 >= 80%

## Nivel 3: Reel Completo

### Bloco 2 (Retencao)
- [ ] Gap de informacao mantido
- [ ] Tensao narrativa com escalada
- [ ] Sem vale de energia
- [ ] Contexto relevante e conciso

### Bloco 3 (Segundo Hook)
- [ ] Re-engajamento presente (30-40s)
- [ ] Segmenta para ICP
- [ ] Novo conflito ou promessa renovada

### Bloco 4 (Conteudo)
- [ ] Entrega valor real
- [ ] Elemento do Movimento presente (OBRIGATORIO)
- [ ] Metafora/exemplo concreto
- [ ] Nao tecnico demais

### Bloco 5 (Moral + CTA)
- [ ] Gera 1 dos 5 sentimentos climaticos (OBRIGATORIO)
- [ ] Insight transformador
- [ ] CTA com comando claro
- [ ] UM unico CTA
- [ ] Nao vende diretamente
- [ ] Score Nivel 3 >= 80%

## Resultado

- [ ] Score Geral >= 80%: APROVADO
- [ ] Score < 80%: REESCREVER
