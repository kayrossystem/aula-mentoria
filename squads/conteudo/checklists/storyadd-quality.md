# Checklist de Qualidade de StoryAds

## FORMATO DE TEXTO
- [ ] Maximo 3 linhas de texto
- [ ] Texto curto e direto
- [ ] Fonte e tamanho legiveis em mobile
- [ ] Sem blocos de texto densos

## ESTRUTURA NARRATIVA
- [ ] Elemento ANTES presente (situacao inicial/dor)
- [ ] Elemento DEPOIS presente (transformacao/resultado)
- [ ] Elemento INSINUACAO presente (curiosidade sobre o como)
- [ ] Progressao Antes -> Depois -> Insinuacao clara

## APARENCIA VISUAL
- [ ] Imagem parece casual/acidental
- [ ] Nao parece producao profissional forcada
- [ ] Parece conteudo organico de Stories
- [ ] Sem elementos graficos excessivos (setas, emojis, badges)

## INTENCIONALIDADE
- [ ] Nao parece intencional/planejado
- [ ] Passa como conteudo espontaneo
- [ ] Sem linguagem de vendas explicita
- [ ] Sem urgencia artificial ("ULTIMAS VAGAS", "SO HOJE")

## OBJETIVO
- [ ] Nao vende diretamente
- [ ] Gera desejo genuino
- [ ] Cria curiosidade sobre o proximo passo
- [ ] Deixa o espectador querendo mais informacao

## TESTE FINAL
- [ ] Voce passaria reto se visse nos Stories? (deve responder NAO)
- [ ] Parece story de um amigo compartilhando algo real?
- [ ] Gera vontade de responder ou clicar?
- [ ] Funciona sem som (para Stories sem audio)?
