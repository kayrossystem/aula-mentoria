# Checklist: Regras de Conteudo

> Checklist rapido para qualquer formato de conteudo.

---

## Tom e Linguagem

- [ ] Tom NUCLEO Torriani (provocativo, direto, sem piedade)
- [ ] Foco no leitor ("voce"), nunca "eu/nos"
- [ ] Sem cliches proibidos
- [ ] Sem palavras banidas
- [ ] Sem frases de rejeicao imediata
- [ ] Vocabulario permitido: rompe, ativa, condena, destrava, instala, acelera

## Estrutura

- [ ] Tipo de post/formato definido
- [ ] Intencao clara (Atracao/Consciencia/Aquecimento/Venda)
- [ ] Progressao emocional presente
- [ ] Tensao do inicio ao fim
- [ ] Solucao nao revelada cedo

## Hook/Abertura

- [ ] Contraintuitivo ou disruptivo
- [ ] Sem perguntas (carrosseis) / Max 5 palavras (reels)
- [ ] 3 variacoes geradas (carrosseis)
- [ ] Zero contexto, so tensao

## CTA

- [ ] Comando moral (nao pedido)
- [ ] Palavra-chave memoravel
- [ ] Promessa de entrega
- [ ] UM unico CTA por peca
- [ ] Sem genericos ("clique aqui", "me chama")

## Engagement e Storytelling

- [ ] Conteudo conta uma historia (nao apenas informa)
- [ ] Existe um arco narrativo: situacao → conflito → resolucao
- [ ] Usa exemplos concretos e especificos (nao generico)
- [ ] Provoca identificacao imediata ("isso sou eu")
- [ ] Cada paragrafo faz o leitor querer ler o proximo (micro-tensoes)

## Alinhamento com Audiencia

- [ ] Linguagem compativel com o nivel de consciencia do publico
- [ ] Dor/desejo abordado e real e verificado (nao inventado)
- [ ] Referencia situacoes do dia-a-dia da audiencia
- [ ] Nao assume conhecimento previo que a audiencia nao tem

## Consistencia Visual e de Formato

- [ ] Formatacao segue o padrao do canal (carrossel/reels/stories/post)
- [ ] Espacamento e quebras de linha facilitam leitura rapida
- [ ] Hierarquia visual clara (titulo > subtitulo > corpo)

## Validacao

- [ ] Passou pelo Oraculo (score >= 80%)
- [ ] Pelo menos 1 dos 5 efeitos obrigatorios
- [ ] Gera desconforto (se nao causa, nao funciona)
- [ ] Lido em voz alta sem soar artificial ou robotico
- [ ] Testado no formato final (preview no celular/plataforma)
- [ ] Nenhuma promessa que nao pode ser sustentada com prova
