# Checklist de Execucao de Estrategia

## COLETA DE INFORMACOES (4 obrigatorias)
- [ ] Mercado/nicho do cliente identificado
- [ ] Persona principal definida (dores, desejos, linguagem)
- [ ] Objetivo da campanha claro (awareness, engajamento, conversao)
- [ ] Restricoes e preferencias do cliente documentadas

## ESTRATEGIA
- [ ] Estrategia correta identificada para o objetivo
- [ ] Justificativa da escolha documentada
- [ ] Metricas de sucesso definidas
- [ ] Riscos mapeados e mitigacoes planejadas

## CRONOGRAMA
- [ ] Cronograma completo com datas e horarios
- [ ] Frequencia de postagem definida
- [ ] Distribuicao equilibrada entre formatos
- [ ] Datas de revisao/ajuste previstas

## TEMPLATES
- [ ] Templates prontos para cada dia do cronograma
- [ ] Cada template segue o padrao do formato (carrossel, reel, stories)
- [ ] Copys completas com hooks, corpo e CTA
- [ ] Briefings visuais incluidos quando necessario

## INTEGRACAO FEED + STORIES
- [ ] Conteudo de Feed e Stories se complementam
- [ ] Stories referenciam/amplificam posts do Feed
- [ ] Progressao narrativa coerente entre formatos
- [ ] Horarios de publicacao coordenados

## TOM E LINGUAGEM
- [ ] Tom imperial consistente em todas as pecas
- [ ] Sem linguagem generica ou cliche
- [ ] Vocabulario alinhado com a marca
- [ ] Autoridade presente sem arrogancia

## CTAs
- [ ] Cada peca tem CTA claro
- [ ] CTAs contem palavra-chave relevante
- [ ] Variacao de CTAs entre pecas (nao repetitivo)
- [ ] CTAs direcionam para acao mensuravel

## VALIDACAO FINAL
- [ ] Validado pelo oraculo
- [ ] Feedback incorporado
- [ ] Versao final aprovada
- [ ] Pronto para execucao
