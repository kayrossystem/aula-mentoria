# Checklist de Qualidade da Bio Imperial

## ESTRUTURA (3 fases obrigatorias)

### Fase 1: Inversao
- [ ] Subverte a expectativa do leitor
- [ ] Primeira frase causa estranhamento
- [ ] Quebra o padrao "ajudo X a fazer Y"

### Fase 2: Desafio
- [ ] Provoca o leitor a questionar suas crencas
- [ ] Cria tensao intelectual ou emocional
- [ ] Posiciona o autor como diferente do mercado

### Fase 3: Redefinicao
- [ ] Redefine o que o autor faz de forma unica
- [ ] Deixa claro o territorio que ocupa
- [ ] Fecha com autoridade, nao com pedido

## REGRAS DE FORMATO
- [ ] Frase da Bio <= 150 caracteres
- [ ] Concisa e impactante (sem enrolacao)
- [ ] Legivel em tela mobile

## PALAVRAS PROIBIDAS
- [ ] Zero uso de "ajudo"
- [ ] Zero uso de "inspiro"
- [ ] Zero uso de "ensino"
- [ ] Nenhum verbo passivo/servil

## CRITERIOS DE QUALIDADE
- [ ] Nao parece bio de LinkedIn
- [ ] Gera tensao ou curiosidade imediata
- [ ] Verbo forte utilizado (comandar, construir, destruir, revelar)
- [ ] Verdade contraintuitiva presente
- [ ] Diferencia claramente de qualquer concorrente
- [ ] Funciona sozinha (sem precisar ver o perfil)

## TESTE FINAL
- [ ] Alguem lendo pela primeira vez ficaria intrigado?
- [ ] Comunica autoridade sem listar credenciais?
- [ ] E memoravel (fica na cabeca)?
